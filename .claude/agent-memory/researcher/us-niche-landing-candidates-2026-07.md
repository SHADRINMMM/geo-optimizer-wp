Оператор (2026-07-07) попросил выбрать 2-3 US-ниши под тестовые лендинги (паттерн `/for-<niche>` + бесплатный AI-score). Полный отчёт с URL-источниками отдан в ответе агента и сохранён в scratchpad сессии (`/tmp/.../us-niches-2026-07-07.md` — эфемерно, не для повторного чтения).

**Итог отбора (топ-3, по силе пруфа + окну конкуренции):**
1. **Dental practices** — старт. Пруф сильный (несколько агентств уже пишут "AI SEO/GEO for dentists" 2026, есть цифры типа "70%+ юзеров исследуют дентал через AI" из агентских блогов — вендор-клейм, не строгий survey). Платформенный клин: PatientPop/Tebra/Weave/Podium управляют онлайн-присутствием практики — стандартизируемая точка входа для авто-фиксов. Дистрибуция: Dental Town forum, r/dentistry, AADOM, ADA/SmileCon.
2. **Med spa/aesthetics** — вторая волна. Похожий угол, выше тикет ($1500-6000/год LTV пациента, рынок $17-18B 2026, AmSpa данные), но платформа сайта менее консолидирована (Squarespace/Wix/WP вперемешку, нет доминирующей PatientPop-аналогии). Жёсткий факт из 5W Research: топ-25 брендов аэстетики контролируют ~95% AI citation share, независимые médspa (11,500+) делят <5% — это и есть рыночная боль, которую продавать. Дистрибуция: AmSpa, Facebook-группы owners, Medical Spa Show (Vegas, апрель).
3. **Personal injury / family law firms** — самый высокий потолок LTV клиента (contingency fee, cost per signed case $2.5-3K против fee $40K+), но конкуренция за нарратив "AI visibility for lawyers" растёт быстрее — крупные legal-marketing агентства (Scorpion, Clio blog, First Page Sage, Esquire Digital, Red Olive, BigDog ICT) уже продают GEO-услугу под этим именем в 2026. Окно уже начало закрываться. Если делать — узкая специализация (PI, не "law firms" вообще). Дистрибуция: PILMMA (ассоциация маркетинга именно для PI-юристов) — сильная точка входа.

**Анти-лист (отложить — недостаточный пруф на паттерн "спрашивают AI за рекомендацией", не general демand):** HVAC/roofing (эмоционально срочный локальный спрос идёт больше через Google Maps здесь-и-сейчас, хотя тренд "45% consumers use AI to find contractor" подтверждён — граница между "ищут инфо" и "просят конкретную рекомендацию" размыта), veterinary clinics (рынок AI-цитирований в vet доминируют корпоративные бренды Mars Petcare/Banfield/VCA/BluePearl — ~третья часть цитирований, независимым тяжело конкурировать даже с продуктом), private schools (спрос на AI-рекомендации не подтверждён, родители пока используют Niche.com/GreatSchools), real estate agents (рынок маркетинга агентам перегрет, хотя пруф спроса сильный — 91% агентов "invisible to AI" по FlyDragon 2026 benchmark, 61.3% buyer-side поисков начинаются в AI).

**Важный методологический момент**: почти все цифры вида "70% patients используют AI для поиска дантиста" идут из блогов маркетинговых агентств самой нишах (vendor-claim), а не из independent survey с методологией. Держать в лендинг-копи с осторожностью или искать первичный источник перед публикацией конкретных %.
